## Data Science Project Template

A **goto template** for starting any data science project with a consistent, well-organized structure. Instead of rebuilding the same scaffold from scratch every time, this template provides a proven layout that works across exploratory analysis, ML experiments, and production pipelines.

### What's Included

- **Standardized directory structure**: Clear separation between raw data, processed data, notebooks, source code, and outputs.
- **Dependency management**: Pre-configured environment files to make setup reproducible across machines.
- **Reproducibility by default**: Structure encourages keeping raw data untouched, logging experiment parameters, and separating data ingestion from analysis.

### Benefits

- Spend less time on project setup, more time on the actual problem.
- Consistent structure makes it easy to pick up any project after a break.
- Shareable and collaborative — teammates know exactly where to look for things.

This template reflects lessons learned from real data science projects where disorganized structure slowed down iteration and made reproducibility painful.
